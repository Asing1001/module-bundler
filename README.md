# Build a module bundler from scratch

## Prerequsite

NodeJS - https://nodejs.org/
Git - https://git-scm.com/downloads

## Getting start

```bash
git clone https://github.com/Asing1001/module-bundler.git
```

## Development

```bash
# Run webpack by the webpack.config.js
npm run webpack

# Run module bundler
npm start
```
